# 🛒 Skyline Store: Printing & Design Solutions

**Skyline Store** is a premium mobile e-commerce platform built with **React Native** and **TypeScript**. It represents the digital evolution of a professional design and printing group with a legacy starting in 2000 from Aleppo to Gaziantep.

---

## 🌟 Legacy & Vision
* **Experience:** Over 24 years of expertise in the modern printing industry.
* **Presence:** Established in Aleppo (2000) and expanded to Gaziantep, Turkey.
* **Mission:** Bridging traditional printing quality with modern mobile commerce.

## 🚀 Technical Tech Stack
* **Framework:** React Native (Expo)
* **Language:** TypeScript (Type-safe development)
* **Backend:** Firebase (Authentication & Real-time Database)
* **Architecture:** Modular Clean Architecture (Services, Hooks, and Context layers)

## 📦 Project Structure
* **`app/`**: Core application screens and navigation logic.
* **`components/`**: Reusable UI components.
* **`config/`**: Firebase and environment configurations.
* **`context/`**: Global state management.
* **`hooks/`**: Custom React hooks for shared logic.
* **`services/`**: API and database interaction layers.

## 🛠️ Key Features
* **Authentication:** Secure user login/signup via Firebase.
* **Product Catalog:** Browsing high-quality printing services and products.
* **Shopping Cart:** Full commerce flow from selection to checkout.
* **Cross-Platform:** Single codebase for both **Android** and **iOS**.
