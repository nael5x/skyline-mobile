# 🛒 Skyline Store: Printing & Design Solutions

Skyline Store is a cross-platform mobile e-commerce platform built with **React Native** and **TypeScript**. This project is currently under active development and focuses on providing a seamless, high-performance digital shopping experience for printing and design services using modern mobile development practices.

## 🚀 Tech Stack

* **Framework:** React Native (Expo)
* **Language:** TypeScript (Type-safe development)
* **Backend:** Firebase (Authentication & Real-time Database)
* **Architecture:** Modular Clean Architecture (Services, Hooks, and Context layers)
* **Localization:** i18next (Multi-language & RTL/LTR support)

## 📦 Project Structure

* `app/`: Core application screens and navigation logic.
* `components/`: Reusable UI components.
* `config/`: Firebase and environment configurations.
* `context/`: Global state management.
* `hooks/`: Custom React hooks for shared logic.
* `services/`: API and database interaction layers.

## 🛠️ Key Features

* **Multi-language Support:** Fully localized for Arabic, Turkish, and English with seamless layout transitions.
* **Authentication:** Secure user login/signup via Firebase.
* **Product Catalog:** High-performance browsing for printing services and products.
* **Shopping Cart:** Full commerce flow from selection to checkout state management.
* **Cross-Platform:** Single unified codebase for both Android and iOS.
